基础设置

1.横竖屏适配

在General界面的Deployment Info选项中
Device Orientation中，选了横屏游戏，就不能勾选竖屏相关方向的选项。会影响sdk界面方向，目前sdk不支持横竖屏自动变换。

如果横屏和竖屏要相互支持，需要在appdelegate文件加上如下代码:
>

#if __IPAD_OS_VERSION_MAX_ALLOWED >= __IPAD_6_0
- (NSUInteger)application:(UIApplication *)application  supportedInterfaceOrientationsForWindow:(UIWindow *)window
{
    return UIInterfaceOrientationMaskAll;
}

#endif

并且在启动时候根据横竖屏设置如下：
竖屏添加的代码：
>
[YeahSDK initSDK:YWOrientationType_Portrait];

横屏添加的代码：
>
[YeahSDK initSDK:YWOrientationType_Landscape];

2.C++和Object-C联编问题
在Build Setting中，other Linker Flags 添加-ObjC

3.跳转回来游戏时，在以下的三个接口的添加[YeahSDK handleOpenURL:url];
>
-(BOOL)application:(UIApplication )application handleOpenURL:(NSURL )url
{
return [YeahSDK handleOpenURL:url];
}

- (BOOL)application:(UIApplication *)app openURL:(NSURL *)url options:(NSDictionary<NSString *,id> *)options
{
    return [YeahSDK handleOpenURL:url];
}

- (BOOL)application:(UIApplication *)application openURL:(NSURL *)url sourceApplication:(NSString *)sourceApplication annotation:(id)annotation
{
    return [YeahSDK handleOpenURL:url];
}

平台信息设置

Activity onCreate方法中，依次添加以下所要功能的代码:

1.添加平台信息设置:
>
[YeahSDK setAppId:@”0” //appKey
appSecret:@”0” //appSecret
platform:@”0” //平台标示
channel:@”0” //渠道ID
gateway:@”https://games.yeahworld.com/“]; //目前固定数值，如果可以可设置热更部分修改

2.添加第三方SDK信息设置(该部分接口在目前版本已经停止使用):
>
[YeahSDK setWeChatAppId:@”0” appSecret:@”0”];
[YeahSDK setWeiboAppId:@”0” appSecret:@”0”];
[YeahSDK setQqAppId:@”0” appSecret:@”0”];
[YeahSDK setAlipayScheme:@”0”];

用户登录

1.开始登录前的准备:
-在监听登录回调的类上，添加YWUserAgencyDelegate协议；

-添加登录回调的代理
>
[[YWUserAgency sharedInstance] setDelegate:self];

-控制登录方式，目前默认设置如下代码，不支持开启其他的登录方式
>
[YeahSDK setSupportLogin:[NSMutableSet setWithObjects:@”guest”, @”user”, nil]];

-YeahSDK标示控制, 默认如下代码
>
[YeahSDK showYWDisplay:YES];

2.登录接口
>
[[YWUserAgency sharedInstance] showUserCenterDialog];

3.用户登录信息回调接口:
>

- (void)userLoginSucceed:(int)userId userName:(NSString *)userName sessionId:(NSString *)sessionId accountType:(int)accountType
{
    NSLog(@"userId %d userName %@ sessionId %@ accountType %d", userId, userName, sessionId, accountType);
    //userId目前停止使用，用userName做真正的userID，并且用sessionId去和php服务器验证
}

用户支付

1.开始支付前的准备:

-在监听支付回调的类上，添加YWPayAgencyDelegate协议；
-添加支付回调的代理
>
[[YWPayAgency sharedInstance] setDelegate:self];
-控制支付方式，目前默认设置如下代码，不支持开启其他的支付方式
>
//unionpay tenpay phone game alipay wechat
[YeahSDK setSupportPay:[NSMutableSet setWithObjects:@”phone”, @”alipay”, @”tenpay”, @”game”, @”unionpay”, @”wechat”, nil]];

2.支付接口
>
//参数分别是AppId是appid，productId商品Id，specifyOrderType针对特别修改订单类型，attach额外字段，iapHandler：游戏商接入设置nil即可
[[YWPayAgency sharedInstance] showCenterDialogWithAppId:@”9527” productId:@”com.yeahworld.yscss.6” specifyOrderType:@”1” attach:@”0” iapHandler:nil];

3.用户支付信息回调接口:
>

- (void)transcationFinished:(BOOL)succeed
{
  NSLog(@"transcationFinished %d", succeed);
}

信息上传

1.在启动应用位置，上传信息接口:
>
[[YWUserAgency sharedInstance] submitStartInfo];

2.用户信息上传接口，在游戏角色登录成功之后调用:
>
[[YWUserAgency sharedInstance] submitUserInfo:@”mojiarongkf” //userId：
userName:@”mojiarongkf” //userName ：备注userId和userName都是用sdk返回userName
serverId:@”1” //服务器Id，默认值1，不需要修改
serverName:@”外服” //服务器名字，默认值外服，如果有相应的名字，可填写
roleId:@”20011988” //游戏角色Id
roleName:@”滕念仪” //游戏角色Id
roleLevel:@”0” //游戏角色等级，默认值0
vipLevel:@”0” //游戏角色vip等级，默认值0
power:@”0” //角色综合战力值,默认值:0
career:@”0” //职业ID,默认值:0
sex:@”0” //性别,默认值:0
realm:@”0” //阵营,默认值:0
gold:@”0”]; //剩余游戏非绑定元宝或钻石数量,默认值:0

3.IAP信息上传接口：

IAP信息需要在角色登录之前上传

//productDict格式

// nil]
NSDictionary* iapDict = [NSDictionary dictionaryWithObjectsAndKeys:
@”600#1#600钻”, @”com.yeahworld.yscss.6”,
@”3000#1#3000钻”, @”com.yeahworld.yscss.30”,
@”6800#1#6800钻”, @”com.yeahworld.yscss.68”,
@”6800#10#6800钻”, @”com.yeahworld.yscss.69”,
@”12800#1#12800钻”, @”com.yeahworld.yscss.128”,
@”25800#1#25800钻”, @”com.yeahworld.yscss.258”,
@”48800#11#48800钻”, @”com.yeahworld.yscss.488”,
@”64800#1#64800钻”, @”com.yeahworld.yscss.648”,
nil];
[YeahSDK setIAPInfo:iapDict];
